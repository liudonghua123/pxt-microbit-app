Create a variable named "hand" - this will help us keep track of whether we have a Rock, Paper or Scissors in our hand. ## {Step 4}

Now add an ``||logic:else if||`` clause to check if the hand value is **2**. In that case, use the ``||basic:show icon||`` function to show a large square icon representing üìÉ Paper.

## {Step 5}
